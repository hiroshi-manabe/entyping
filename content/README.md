# Content Packages

This directory contains content packages that the typing app can load.

Current packages:

- `new_crown1`
  - current runtime asset area for `NEW CROWN 1`
- `jhs1_typing_500`
  - supplemental 500-sentence JHS 1 typing content

The goal is for content to be replaceable without restructuring the typing program itself.

If source analysis or reverse-engineering work is still messy or provisional, keep it under `workspace/` until it is ready to become a runtime content package.
